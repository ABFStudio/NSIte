# NSIte
Site de présentation de la spécialité NSI

## Windows
```
git clone -b nsite-flask https://github.com/NSIte.git nsite-flask
cd nsite-flask
python -m venv .env
.env\scripts\Activate.ps1
pip install -r requirements.txt
python main.py
```

## macOS/Linux
```
git clone -b nsite-flask https://github.com/NSIte.git nsite-flask
cd nsite-flask
python3 -m venv .env
source .env/scripts/activate
pip install -r requirements.txt
python main.py
```
